# Mister White

Juego de fiesta en Next.js 15 + React 19 + Tailwind v4 + lucide-react.

## UI: mobile-first

Se juega **pasando el teléfono** entre jugadores. Diseña siempre desde móvil:
touch targets ≥44px, una sola columna, texto legible a un brazo de distancia.
Tailwind v4 con spacing fluido (`clamp`).
